**核心代码：**
```python
def identify_search_field(fields):
    # 策略1: 单一字段
    if len(fields) == 1:
        return fields[0]['name']
    
    # 策略2: 常见字段名
    common_names = ['q', 'wd', 'query', 'search', 'key', 'keyword']
    for field in fields:
        if field['name'] in common_names:
            return field['name']
    
    # 策略3: 字段名包含关键词
    for field in fields:
        if 'search' in field['name'].lower():
            return field['name']
    
    # 策略4: 文本类型且无默认值
    for field in fields:
        if field['type'] == 'text' and not field['value']:
            return field['name']
    
    return None
```

**使用方法：**
```bash
cd tools
python quick_search_url_extractor.py https://www.example.com
```

### 3. 工作流程优化

#### 新工作流程

```
阶段1：收集信息
├─ 步骤1：快速尝试获取搜索地址（优先）
│   ├─ 使用 quick_search_url_extractor.py
│   ├─ 成功？ → 步骤2：验证搜索地址
│   └─ 失败？ → 步骤3：深度分析
├─ 步骤2：验证搜索地址（如果步骤1成功）
│   ├─ 浏览器测试
│   ├─ 对比验证
│   └─ 成功 → 进入阶段2
├─ 步骤3：深度分析（如果步骤1失败或验证失败）
│   ├─ 使用 quick_analyze.py
│   ├─ 手动分析
│   └─ 进入阶段2
```
