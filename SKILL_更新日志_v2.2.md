### 3. 更新现有文档

#### WORKFLOW.md

**主要更新：**
- ✅ 重组阶段1流程
- ✅ 新增步骤1：快速搜索地址提取
- ✅ 新增步骤2：验证搜索地址
- ✅ 新增步骤3：深度分析（备选）
- ✅ 更新章节编号

**新流程：**
```
阶段1：收集信息
├─ 步骤1：快速尝试获取搜索地址（优先）
│   ├─ 使用 quick_search_url_extractor.py
│   ├─ 成功？ → 步骤2：验证搜索地址
│   └─ 失败？ → 步骤3：深度分析
├─ 步骤2：验证搜索地址（如果步骤1成功）
│   ├─ 浏览器测试
│   ├─ 对比验证
│   └─ 成功 → 进入阶段2
├─ 步骤3：深度分析（如果步骤1失败或验证失败）
│   ├─ 使用 quick_analyze.py
│   ├─ 手动分析
│   └─ 进入阶段2
```

**优势：**
- ⚡ 更快（优先快速方法）
- 🎯 更准（多层级验证）
- 🔄 更灵活（自动切换）
