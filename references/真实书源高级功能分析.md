## 功能1：jsLib - JavaScript库

### 概述

jsLib是Legado提供的JavaScript库功能，可以在书源中定义自定义函数，供其他规则调用。

使用场景：
- 图片解密
- 数据压缩/解压
- 加密算法
- 自定义工具函数

### 实例1：喜漫漫画 - AES解密

功能：使用AES/CBC/PKCS5Padding解密图片

关键点：
- jsLib定义decode函数
- 使用AES加密算法
- coverDecodeJs在封面解密时调用
- imageDecode在正文图片解密时调用

### 实例2：3A小说 - LZString压缩解密

功能：使用LZString算法解压数据

关键点：
- jsLib定义XSVUE对象
- decompressFromBase64解压数据
- 在init、bookList、content中使用

### jsLib使用规范

定义位置：书源JSON的顶层字段

调用方式：
1. coverDecodeJs：封面解密
2. imageDecode：正文图片解密
3. @js:或<js></js>：在规则中调用

注意事项：
- jsLib中可以使用Java类
- jsLib中可以使用this.java访问Legado的Java API
- jsLib代码较长时需要注意性能

## 功能2：loginCheckJs - 验证处理

### 概述

loginCheckJs是在每次请求后执行的JavaScript代码，可以处理各种验证机制。

使用场景：
- CloudFlare验证
- 滑块验证
- 人机验证
- Cookie管理

### 实例1：霹雳书屋 - 滑块验证

关键点：
- 检测CloudFlare验证（403/503/429状态码）
- 使用WebView自动处理验证
- 检测滑块验证（/_guard/）
- 使用AES加密生成滑块轨迹
- 记录失败次数，限制尝试次数

### 实例2：喜漫漫画 - CloudFlare验证

关键点：
- 检测多种CloudFlare验证模式
- 首次拦截使用WebView自动处理
- 多次拦截使用手动验证
- 清除Cookie重新尝试
- 记录拦截次数

### loginCheckJs使用规范

定义位置：书源JSON的顶层字段

可用的API：
- result.url() - 获取请求URL
- result.body() - 获取响应内容
- result.code() - 获取状态码
- java.webView(url, url, js) - 打开WebView
- java.startBrowserAwait(url, title) - 打开浏览器等待
- java.toast(msg) - 显示提示
- source.get(key) - 获取变量
- source.put(key, value) - 设置变量
- cookie.removeCookie(url) - 删除Cookie

注意事项：
- 必须返回处理后的result
- 避免无限循环
- 记录失败次数，防止重试过多
- 使用日志调试

## 功能3：loginUi - 用户界面

### 概述

loginUi是在书源设置界面显示的用户交互界面。

使用场景：
- 接口切换
- 参数设置
- 功能开关

### 实例：霹雳书屋 - 接口切换

关键点：
- 提供4个接口选项
- 使用set()和get()函数
- 提示用户需要长按发现列表刷新

### loginUi使用规范

定义位置：书源JSON的顶层字段

格式：JSON数组

字段说明：
- name：按钮显示名称
- type：类型（目前支持"button"）
- action：点击后执行的JavaScript代码

注意事项：
- 必须是JSON数组格式
- 使用source.setVariable()保存变量
- 使用source.getVariable()读取变量

function bypassCloudFlare(result) {
    var src = result.body();
    var url = result.url();
    
    var cfPatterns = [
        /Just a moment/,
        /Checking your browser/,
        /DDoS protection/
    ];
    
    var isBlocked = cfPatterns.some(function(pattern) {
        return src.match(pattern);
    });
    
    if (!isBlocked) {
        return result;
    }
    
    // 使用WebView处理
    var webResult = java.webView(url, url, 
        "setTimeout(function() {" +
        "    window.legado.getHTML(document.documentElement.outerHTML);" +
        "}, 10000);"
    );
    
    result = java.connect(url);
    return result;
}

result = bypassCloudFlare(result);
