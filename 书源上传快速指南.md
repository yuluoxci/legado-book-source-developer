## 📝 使用方法

### 命令行上传

```bash
# 基本用法
python upload_book_source.py book_source.json

# 使用自定义配置
python upload_book_source.py book_source.json custom_config.json
```

## 📚 书源格式要求

### 最小必需字段

```json
{
  "bookSourceName": "书源名称",
  "bookSourceUrl": "https://example.com",
  "bookSourceType": 0,
  "searchUrl": "/search?q={{key}}",
  "ruleSearch": {
    "bookList": ".book-item",
    "bookUrl": "a@href"
  },
  "ruleToc": {
    "chapterList": "li",
    "chapterName": "a@text",
    "chapterUrl": "a@href"
  },
  "ruleContent": {
    "content": ".content@text"
  }
}
```

### 必需规则字段

| 规则对象 | 必需字段 |
|---------|---------|
| ruleSearch | bookList, bookUrl |
| ruleToc | chapterList, chapterName, chapterUrl |
| ruleContent | content |

### 自动补充的可选字段

工具会自动添加以下字段（如果缺失）:
- `bookSourceGroup`
- `enabled`
- `enabledExplore`
- `enabledCookieJar`
- `loginUrl`, `loginUi`, `loginCheckJs`
- `concurrentRate`, `header`
- `exploreUrl`
- `ruleBookInfo`, `ruleExplore`, `ruleReview`
- `bookSourceComment`, `variableComment`

## 🛠️ 常见问题

### Q1: 验证失败怎么办?

A: 根据错误提示修改书源:
- 检查必需字段是否完整
- 检查规则对象必需字段是否非空
- 检查字段类型是否正确

### Q2: 警告会影响上传吗?

A: 不会。警告只是提示信息:
- 缺少可选字段 → 自动补充
- 类型问题 → 自动修正
- 建议字段 → 可选添加

### Q3: 如何仅验证不上传?

A: 使用BookSourceValidator类:

```python
from upload_book_source import BookSourceValidator

validator = BookSourceValidator()
is_valid, fixed_source, errors, warnings = validator.validate_and_fix(source)
```

### Q4: 字段类型错误会自动修正吗?

A: 会的:
- `"0"` → `0` (字符串转整数)
- `0` → `"0"` (整数转字符串)

### Q5: 验证通过但上传失败?

A: 检查:
- 网络连接
- 上传服务状态
- 上传配置是否正确
