# 书源验证与上传功能说明

## 功能概述

`upload_book_source.py` 工具现已集成书源验证功能,在上传前会自动:
1. **验证书源格式** - 检查是否符合Legado标准
2. **自动修正问题** - 尝试修正格式错误和补充缺失字段
3. **显示验证结果** - 清晰展示验证通过/失败、警告和错误信息
4. **智能上传** - 只有验证通过的书源才会上传

## 主要改进

### 1. 新增 BookSourceValidator 类

负责验证书源JSON格式,包括:

**必需字段验证:**
- `bookSourceUrl` - 书源URL
- `bookSourceName` - 书源名称
- `bookSourceType` - 书源类型 (0=文本, 1=音频, 2=图片)
- `searchUrl` - 搜索URL
- `ruleSearch` - 搜索规则对象
- `ruleToc` - 目录规则对象
- `ruleContent` - 正文规则对象

**规则字段验证:**
- `ruleSearch.bookList` - 书籍列表选择器 (必需)
- `ruleSearch.bookUrl` - 书籍URL选择器 (必需)
- `ruleSearch.name` - 书名选择器 (推荐)
- `ruleToc.chapterList` - 章节列表选择器 (必需)
- `ruleToc.chapterName` - 章节名选择器 (必需)
- `ruleToc.chapterUrl` - 章节URL选择器 (必需)
- `ruleContent.content` - 正文选择器 (必需)

**可选字段自动补充:**
- `bookSourceGroup` - 书源分组
- `enabled` - 是否启用
- `enabledExplore` - 是否启用发现
- `enabledCookieJar` - 是否启用Cookie
- `loginUrl` - 登录URL
- `loginUi` - 登录界面
- `loginCheckJs` - 登录检查脚本
- `concurrentRate` - 并发率
- `header` - 请求头
- `exploreUrl` - 发现URL
- `ruleBookInfo` - 书籍信息规则
- `ruleExplore` - 发现规则
- `ruleReview` - 评论规则
- `bookSourceComment` - 书源注释
- `variableComment` - 变量注释

### 2. 自动修正能力

**类型转换:**
- 字符串到整数: `"0"` → `0`
- 整数到字符串: `0` → `"0"`

**字段补充:**
- 自动添加缺失的可选字段及其默认值
- 自动为缺失的必需规则对象创建空字典 `{}`

### 3. 验证流程

```
用户输入书源
    ↓
验证书源格式
    ↓
发现问题?
    ├─ 是 → 尝试自动修正
    │         ↓
    │     修正成功?
    │         ├─ 是 → 显示警告 → 继续上传
    │         └─ 否 → 显示错误 → 终止上传
    └─ 否 → 验证通过 → 继续上传
```

## 使用方法

### 1. 命令行使用

```bash
# 上传书源文件（自动验证）
python upload_book_source.py book_source.json

# 上传书源文件（使用自定义配置）
python upload_book_source.py book_source.json custom_config.json
```

### 2. 代码中使用

```python
from upload_book_source import BookSourceUploader

uploader = BookSourceUploader()

# 上传书源（自动验证并修正）
success, result = uploader.upload_book_source(book_source_json)

if success:
    print(f"上传成功: {result['result']['download_url']}")
else:
    print(f"上传失败: {result['error']}")
```

### 3. 仅验证不上传

如果只需要验证书源格式而不上传:

```python
from upload_book_source import BookSourceValidator

validator = BookSourceValidator()

# 验证并修正书源
is_valid, fixed_source, errors, warnings = validator.validate_and_fix(book_source_json)

if is_valid:
    print("✅ 验证通过")
    # 使用 fixed_source（已修正的书源）
else:
    print("❌ 验证失败")
    print(f"错误: {errors}")
    print(f"警告: {warnings}")
```

## 注意事项

### 1. 验证规则

- **验证通过**: 书源可以正常使用,会执行上传
- **验证失败但可修正**: 自动修正后显示警告,继续上传
- **验证失败且无法修正**: 显示错误,终止上传

### 2. 错误处理

以下情况会导致验证失败:
- 缺少 `bookSourceName` 或 `bookSourceUrl`
- `ruleSearch.bookList` 或 `ruleSearch.bookUrl` 为空
- `ruleToc` 中缺少必需字段
- `ruleContent.content` 为空

以下情况会自动修正:
- 缺少可选字段（自动添加默认值）
- 字段类型错误（字符串↔整数转换）
- 空规则对象（自动创建 `{}`）

### 3. 警告提示

以下情况会产生警告:
- 缺少推荐字段（如 `ruleSearch.name`）
- 自动添加了可选字段
- 自动修正了字段类型
